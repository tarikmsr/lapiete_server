# Vanilla Nodejs RestFul CRUD API Project 
> This is vanilla Node.js CRUD API movie project.

## Routes
```bash

GET      /api/form
GET      /api/form/:id
GET      /api/user/:id

POST     /api/form
POST     /api/login

PUT      /api/form/:id
DELETE   /api/form/:id

```

## Installation

```bash
# Install dependencies
npm install

# Run in develpment
npm run dev

# Run in production
npm start
```
